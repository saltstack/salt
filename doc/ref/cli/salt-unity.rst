==============
``salt-unity``
==============

A unified invocation wrapper around other Salt CLI scripts.

Synopsis
========

.. code-block:: bash

    salt-unity salt '*' test.ping

Description
===========

This script takes an argument which is one of the other Salt CLI scripts and
invokes that script.
